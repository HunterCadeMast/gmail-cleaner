# Gmail Cleaner

A simple Node.js script that automatically moves **non-starred Gmail emails** to Trash using the Gmail API.

⚠️ **Please use with caution...**

---

## Features

- Uses the official **Gmail API**.
- OAuth2 authentication (Google sign-in).
- Trashes **non-starred emails**.
- Avoids re-processing the same messages.
- Progress logging while running.

---

## Requirements

- **Node.js** v18+ (recommended)
- A **Google account**
- Gmail API enabled in Google Cloud Console

---

## Installation

To install, run the script and create a project on Google Cloud with the credentials.js file to authorize access to Gmail.
